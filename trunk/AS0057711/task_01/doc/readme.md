## Task 1. Modeling controlled object

### Описание
В рамках первой задачи была реализована программа на языке C++, моделирующая температуру объекта управления. Объект описывается дифференциальным уравнением, которое было преобразовано в две дискретные математические модели:

1. **Линейная модель:** $y_{\tau+1} = a y_\tau + b u_\tau$
2. **Нелинейная модель:** $y_{\tau+1} = a y_\tau - b y_{\tau-1}^2 + c u_\tau + d \sin(u_{\tau-1})$

### Реализация
Исходный код программы располагается в каталоге `src/`. 
Программа инициализирует константы ($a, b, c, d$) и начальные условия ($y_0$, $y_{-1}$), после чего в цикле рассчитывает значения температуры на $n$ дискретных моментов времени. Результаты работы для обеих моделей выводятся в консоль в табличном виде для сравнительного анализа.
