## Цель работы
Написать программу на языке C++, которая имитирует изменение температуры управляемого объекта на основе линейной и нелинейной дискретных моделей.

## Описание математической модели

Исходное дифференциальное уравнение объекта:
dy(τ)/dτ = u(τ)/C + (Y0 - y(τ))/RC

После дискретизации были получены следующие модели:
1. **Линейная:** y_{τ+1} = a*y_{τ} + b*u_{τ}
2. **Нелинейная:** y_{τ+1} = a*y_{τ} - b*(y_{τ-1})^2 + c*u_{τ} + d*sin(u_{τ-1})

## Описание реализации
Код написан на C++. Для хранения значений во времени использованы структуры `std::vector`. 
В программе задаются начальные условия (например, начальная комнатная температура) и постоянный входной тепловой сигнал. Программа в цикле рассчитывает значения для заданного количества дискретных шагов времени и выводит результат в консоль.
