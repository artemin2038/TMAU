# Лабораторная работа №1 — Моделирование объекта управления

**Выполнил:** Арсений Юрковский, гр. АС-577

## Цель
Разработать программу на C++ для симуляции температуры управляемого объекта с использованием линейной и нелинейной дискретных моделей, полученных из исходного дифференциального уравнения.

## Теоретические сведения
Уравнение, описывающее тепловой процесс объекта:

$$\frac{dy(\tau)}{d\tau} = \frac{u(\tau)}{C} + \frac{Y_0 - y(\tau)}{RC}$$

После его дискретизации были выведены две расчётные модели:

1. Линейная модель:
   $$y_{\tau+1} = a \cdot y_\tau + b \cdot u_\tau$$

2. Нелинейная модель:
   $$y_{\tau+1} = a \cdot y_\tau - b \cdot y_{\tau-1}^2 + c \cdot u_\tau + d \cdot \sin(u_{\tau-1})$$

### Обозначения
- $\tau$ — дискретный момент времени;
- $y(\tau)$ — температура объекта;
- $u(\tau)$ — управляющее тепловое воздействие;
- $Y_0$ — температура окружающей среды;
- $a, b, c, d$ — параметры модели.

## Выполнение
Код написан на языке C++ (стандарт C++17). Программа последовательно вычисляет значения температуры на 15 шагах для обеих моделей. Для сборки используется CMake. Результаты выводятся в виде таблицы в консоль.

В проекте изменены параметры относительно базового варианта:
- $a = 0.9$, $b_{lin} = 0.35$, $b_{nonlin} = 0.015$, $c = 0.35$, $d = 0.25$;
- начальная температура $Y_0 = 22^\circ$;
- входной сигнал $u = 10$ (постоянный).

## Вывод
В ходе работы написана рабочая программа, корректно моделирующая динамику температуры для обеих моделей. Линейная модель даёт плавный рост температуры, нелинейная — более сложное поведение благодаря квадратичной и тригонометрической составляющим. Сборка через CMake выполнена успешно.
