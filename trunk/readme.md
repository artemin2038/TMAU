### Цель работы
Разработать программу на C++ для моделирования температуры управляемого объекта с использованием линейной и нелинейной моделей.

### Теоретическая часть
Объект управления описывается дифференциальным уравнением:

$$\frac{dy(\tau)}{d\tau} = \frac{u(\tau)}{C} + \frac{Y_0 - y(\tau)}{RC}$$

где:
- $\tau$ — время
- $y(\tau)$ — входная температура
- $u(\tau)$ — входной сигнал
- $Y_0$ — комнатная температура
- $C, RC$ — константы

Для дискретного моделирования используются две модели:

**Линейная модель:**
$$y_{\tau+1} = a y_{\tau} + b u_{\tau}$$

**Нелинейная модель:**
$$y_{\tau+1} = a y_{\tau} - b y_{\tau-1}^2 + c u_{\tau} + d \sin(u_{\tau-1})$$

### Реализация

#### Структура программы
Программа реализует:
1. Расчет линейной модели
2. Расчет нелинейной модели
3. Сравнение результатов
4. Визуализацию в табличном виде

#### Основные функции
- `inputSignal()` - генерация входного сигнала управления
- `linearModel()` - расчет по линейной модели
- `nonlinearModel()` - расчет по нелинейной модели
- `printResults()` - вывод результатов

#### Параметры модели
- a = 0.8
- b = 0.3  
- c = 0.2
- d = 0.1
- Начальная температура = 20°C
- Количество шагов = 20

### Выводы

В ходе выполнения лабораторной работы была разработана программа для моделирования температуры управляемого объекта. Были реализованы две модели:

1. **Линейная модель** - более простая, использует только текущее значение температуры и управляющий сигнал. Дает приближенный результат.

2. **Нелинейная модель** - более сложная, учитывает предыдущие значения и нелинейные эффекты (квадрат и синус). Дает более точное приближение к реальному поведению объекта.

Нелинейная модель более точная, но требует больше вычислительных ресурсов и памяти для хранения предыдущих значений.
